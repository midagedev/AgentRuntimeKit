# Support

AgentRuntimeKit is community-maintained.

- Use GitHub Discussions for integration questions and design proposals.
- Use GitHub Issues for reproducible defects and accepted feature work.
- Use GitHub private vulnerability reporting for security or privacy issues.

Please include the Swift and Xcode versions, target platform, selected products,
the smallest reproducible request, and sanitized error category. Never include
API keys, authorization headers, provider request bodies, health data, memory
content, or private tool arguments.
